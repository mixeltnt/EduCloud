INFORME DEL PROYECTO EDUCLOUD
Landing Page responsiva con React, Vite y Bootstrap

1. QUE SE HIZO EN EL PROYECTO

Se transformo la plantilla inicial en una landing page para EduCloud, una academia virtual de cursos online de programacion y diseno.

Cambios principales:
- Se cargo Bootstrap 5 desde el CDN en index.html.
- Se cambio el titulo del sitio a "EduCloud - Cursos online".
- Se creo una estructura por componentes en src/components.
- Se creo un Navbar sticky responsivo con menu hamburguesa y dropdown de cursos.
- Se creo una Hero section con CTA principal para abrir el modal de registro.
- Se creo un carrusel de cursos con useState y useEffect.
- Se creo un modal de registro con validacion de formulario.
- Se creo un acordeon de preguntas frecuentes.
- Se crearon tarjetas de instructores.
- Se creo un footer con newsletter y validacion de correo.
- Se agregaron estilos responsivos en App.css y Navbar.css.
- Se agregaron comentarios en los componentes para explicar las partes importantes.

2. COMO FUNCIONA LA APP

App.jsx es el componente principal. Importa todas las secciones y controla el estado showModal.

showModal:
- false: el modal no se muestra.
- true: el modal de registro aparece.

La funcion setShowModal(true) se pasa como prop a Navbar y Hero con el nombre onRegisterClick.
Eso permite que el boton "Registrarme" del Navbar y el boton "Inscribirme gratis" del Hero abran el mismo modal.

3. HOOKS USADOS

useState:
- Navbar.jsx: controla si esta abierto el menu mobile y si esta abierto el dropdown.
- App.jsx: controla si se muestra el modal.
- CoursesCarousel.jsx: controla el curso activo del carrusel.
- FAQAccordion.jsx: controla que pregunta del acordeon esta abierta.
- RegistrationModal.jsx: controla datos del formulario, errores y mensaje de exito.
- Footer.jsx: controla correo y mensaje del newsletter.

useEffect:
- CoursesCarousel.jsx: crea un intervalo para cambiar automaticamente el curso cada 4.5 segundos.
- Tambien limpia el intervalo cuando el componente se desmonta, usando return () => clearInterval(timer).

4. EXPLICACION DE COMPONENTES

Navbar.jsx:
Muestra la barra superior. Usa clases Bootstrap como navbar, navbar-expand-lg, navbar-toggler y dropdown-menu.
La interactividad no depende del JavaScript de Bootstrap, sino de React con useState.

Hero.jsx:
Es la primera seccion visible. Muestra el nombre EduCloud, texto de presentacion, botones CTA y una tarjeta visual que simula la plataforma.

Features.jsx:
Muestra beneficios del metodo EduCloud. Usa un arreglo llamado features y map para generar tarjetas.

CoursesCarousel.jsx:
Muestra cursos destacados. Tiene botones anterior/siguiente, indicadores y cambio automatico con useEffect.

Instructors.jsx:
Muestra tarjetas de instructores. Usa un arreglo de datos y map.

FAQAccordion.jsx:
Muestra preguntas frecuentes. El estado openIndex indica cual pregunta esta abierta.

RegistrationModal.jsx:
Modal funcional de registro. Valida nombre, email y curso de interes. Si todo esta correcto, muestra confirmacion.

Footer.jsx:
Muestra informacion del sitio, links y formulario de newsletter con validacion de email.

import { useState } from 'react'
import './Navbar.css'

export default function Navbar({ onRegisterClick }) {
  // isOpen controla el menu hamburguesa en pantallas pequenas.
  const [isOpen, setIsOpen] = useState(false)
  // coursesOpen controla el submenu desplegable de cursos.
  const [coursesOpen, setCoursesOpen] = useState(false)

  // Al navegar a una seccion se cierran los menus abiertos.
  const closeMenu = () => {
    setIsOpen(false)
    setCoursesOpen(false)
  }

  return (
    <nav className='navbar navbar-expand-lg edu-navbar sticky-top'>
      <div className='container'>
        <a className='navbar-brand edu-brand' href='#hero' onClick={closeMenu}>
          Edu<span>Cloud</span>

          className='navbar-toggler'
          type='button'
          aria-controls='eduNavbar'
          aria-expanded={isOpen}
          aria-label='Abrir menu'
          onClick={() => setIsOpen(!isOpen)}
        >
          <span className='navbar-toggler-icon'></span>

import { useEffect, useState } from 'react'

// Cada objeto representa una diapositiva del carrusel.
const courses = [
  {
    title: 'Frontend con React',
    category: 'Programacion',
    level: 'Intermedio',
    text: 'Componentes, hooks, consumo de APIs y despliegue con Vite.',
    accent: 'frontend'
  },
  {
    title: 'UX/UI desde cero',
    category: 'Diseno',
    level: 'Inicial',
    text: 'Investigacion, wireframes, prototipos en Figma y pruebas con usuarios.',
    accent: 'design'
  },
  {
    title: 'Backend con Node',
    category: 'Programacion',
    level: 'Intermedio',
    text: 'APIs REST, bases de datos, autenticacion y buenas practicas.',
    accent: 'backend'
  }
]

export default function CoursesCarousel() {
  // Guarda la posicion del curso que se esta mostrando.
  const [activeIndex, setActiveIndex] = useState(0)

  useEffect(() => {
    // Cambia automaticamente de curso cada 4.5 segundos.
    const timer = setInterval(() => {
      setActiveIndex((current) => (current + 1) % courses.length)
    }, 4500)

    // Limpia el intervalo cuando el componente se desmonta.
    return () => clearInterval(timer)
  }, [])

  const goToCourse = (index) => setActiveIndex(index)
  // El modulo evita que el indice se salga del arreglo.
  const prevCourse = () => setActiveIndex((activeIndex - 1 + courses.length) % courses.length)
  const nextCourse = () => setActiveIndex((activeIndex + 1) % courses.length)
  const activeCourse = courses[activeIndex]

  return (
    <section id='courses' className='section-block courses-section'>
      <div className='container'>
        <div className='section-heading'>
          <p className='section-kicker'>Cursos destacados</p>
          <h2>Elige una ruta y avanza con proyectos guiados</h2>
        </div>

        <div className={`course-carousel course-${activeCourse.accent}`}>
          <button className='carousel-control prev' type='button' onClick={prevCourse} aria-label='Curso anterior'>
            {'<'}
          </button>

          <article className='course-slide'>
            <div>
              <span className='badge rounded-pill text-bg-light'>{activeCourse.category}</span>
              <span className='badge rounded-pill text-bg-dark ms-2'>{activeCourse.level}</span>
            </div>
            <h3>{activeCourse.title}</h3>
            <p>{activeCourse.text}</p>
            <a className='btn btn-light' href='#footer'>Recibir programa</a>
          </article>

          <button className='carousel-control next' type='button' onClick={nextCourse} aria-label='Curso siguiente'>
            {'>'}
          </button>
        </div>

        <div className='carousel-indicators-custom' aria-label='Indicadores del carrusel'>
          {/* Indicadores manuales para saltar directamente a un curso. */}
          {courses.map((course, index) => (
            <button
              className={index === activeIndex ? 'active' : ''}
              key={course.title}
              type='button'
              aria-label={`Ver ${course.title}`}
              onClick={() => goToCourse(index)}
            ></button>
          ))}
        </div>
      </div>
    </section>
  )
}

import { useState } from 'react'

// Valores iniciales del formulario. Sirven tambien para limpiar despues de enviar.
const initialForm = {
  name: '',
  email: '',
  interest: ''
}

export default function RegistrationModal({ isOpen, onClose }) {
  // formData guarda lo que el usuario escribe en cada campo.
  const [formData, setFormData] = useState(initialForm)
  // errors guarda los mensajes de validacion por campo.
  const [errors, setErrors] = useState({})
  // submitted cambia la vista del formulario por el mensaje de exito.
  const [submitted, setSubmitted] = useState(false)

  // Valida los campos antes de aceptar el registro.
  const validate = () => {
    const nextErrors = {}

    if (formData.name.trim().length < 3) {
      nextErrors.name = 'Ingresa al menos 3 caracteres.'
    }

    if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(formData.email)) {
      nextErrors.email = 'Ingresa un correo valido.'
    }

    if (!formData.interest) {
      nextErrors.interest = 'Selecciona un curso.'
    }

    return nextErrors
  }

  const handleChange = (event) => {
    const { name, value } = event.target
    // name permite actualizar dinamicamente name, email o interest.
    setFormData({ ...formData, [name]: value })
    setErrors({ ...errors, [name]: '' })
  }

  const handleSubmit = (event) => {
    event.preventDefault()
    const nextErrors = validate()
    setErrors(nextErrors)

    if (Object.keys(nextErrors).length === 0) {
      // Si no hay errores, se muestra el mensaje de confirmacion.
      setSubmitted(true)
      setFormData(initialForm)
    }
  }

  const handleClose = () => {
    // Deja el modal limpio para la siguiente vez que se abra.
    setSubmitted(false)
    setErrors({})
    onClose()
  }

  // Si el modal esta cerrado, no se renderiza nada en pantalla.
  if (!isOpen) return null

  return (
    <>
      <div className='modal fade show edu-modal' tabIndex='-1' role='dialog' aria-modal='true'>
        <div className='modal-dialog modal-dialog-centered'>
          <div className='modal-content'>
            <div className='modal-header'>
              <div>
                <p className='section-kicker mb-1'>Registro</p>
                <h2 className='modal-title'>Comienza en EduCloud</h2>
              </div>
              <button type='button' className='btn-close' aria-label='Cerrar' onClick={handleClose}></button>
            </div>
            <div className='modal-body'>
              {submitted ? (
                <div className='alert alert-success mb-0'>
                  Registro enviado. Te contactaremos con el programa del curso.
                </div>
              ) : (
                <form noValidate onSubmit={handleSubmit}>
                  <div className='mb-3'>
                    <label className='form-label' htmlFor='name'>Nombre completo</label>
                    <input
                      className={`form-control ${errors.name ? 'is-invalid' : ''}`}
                      id='name'
                      name='name'
                      type='text'
                      value={formData.name}
                      onChange={handleChange}
                    />
                    {errors.name && <div className='invalid-feedback'>{errors.name}</div>}
                  </div>
                  <div className='mb-3'>
                    <label className='form-label' htmlFor='email'>Correo electronico</label>
                    <input
                      className={`form-control ${errors.email ? 'is-invalid' : ''}`}
                      id='email'
                      name='email'
                      type='email'
                      value={formData.email}
                      onChange={handleChange}
                    />
                    {errors.email && <div className='invalid-feedback'>{errors.email}</div>}
                  </div>
                  <div className='mb-4'>
                    <label className='form-label' htmlFor='interest'>Curso de interes</label>
                    <select
                      className={`form-select ${errors.interest ? 'is-invalid' : ''}`}
                      id='interest'
                      name='interest'
                      value={formData.interest}
                      onChange={handleChange}
                    >
                      <option value=''>Selecciona una opcion</option>
                      <option value='react'>Frontend con React</option>
                      <option value='uxui'>UX/UI desde cero</option>
                      <option value='node'>Backend con Node</option>
                    </select>
                    {errors.interest && <div className='invalid-feedback'>{errors.interest}</div>}
                  </div>
                  <button className='btn btn-primary w-100' type='submit'>
                    Enviar registro
                  </button>
                </form>
              )}
            </div>
          </div>
        </div>
      </div>
      <div className='modal-backdrop fade show'></div>
    </>
  )
}

6. PREGUNTAS QUE TE PUEDEN HACER Y RESPUESTAS CORTAS

Pregunta: Por que usaste componentes?
Respuesta: Para separar la pagina en partes reutilizables y faciles de mantener: Navbar, Hero, Cursos, Instructores, FAQs, Modal y Footer.

Pregunta: Donde esta la validacion?
Respuesta: En RegistrationModal.jsx para el formulario de registro y en Footer.jsx para el newsletter.

Pregunta: Donde se usa useEffect?
Respuesta: En CoursesCarousel.jsx. Se usa para cambiar automaticamente el curso activo cada 4.5 segundos y limpiar el intervalo.

Pregunta: Como se abre el modal?
Respuesta: App.jsx tiene el estado showModal. Navbar y Hero reciben la funcion onRegisterClick, que ejecuta setShowModal(true).

Pregunta: Como funciona el acordeon?
Respuesta: FAQAccordion.jsx usa openIndex. Si openIndex coincide con el indice de una pregunta, esa respuesta se muestra con la clase show.

Pregunta: Como se hizo responsive?
Respuesta: Se usaron clases Bootstrap como container, row, col-md-4, col-lg-6, navbar-expand-lg y media queries en CSS.
